## Error Contracts

All methods may raise:

| Exception | When |
|-----------|------|
| `AuthenticationError` | Invalid/expired credentials (401) |
| `RateLimitError` | Rate limited (429) — auto-retry with backoff |
| `QueryError` | Invalid parameters (400) |
| `ServerError` | Mixpanel server error (5xx) |
| `WorkspaceScopeError` | No workspace ID configured |
| `APIError` | Any other API error |
